# Tauri Image Viewer (Svelte)

Minimal image viewer built with Tauri + Svelte. Supports drag & drop, file picker, zoom controls, and basic navigation.

## Prerequisites

- Node.js LTS
- Rust (MSVC toolchain)
- Windows C++ Build Tools + Windows SDK

## Install

```bash
npm install
```

## Run (desktop)

```bash
npm run tauri dev
```

## Build (desktop)

```bash
npm run tauri build
```
